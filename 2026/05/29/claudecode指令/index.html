Claude Code 常用 / 指令汇总

适用于 Anthropic Claude Code CLI 开发环境
常见用途：代码生成、项目分析、Git 操作、终端辅助、调试等

1. 基础命令

指令 作用
/help 查看帮助
/clear 清空当前上下文
/exit 退出 Claude Code
/reset 重置会话
/status 查看当前状态
/config 查看配置
/model 切换模型

2. 文件与项目操作

指令 作用
/ls 查看当前目录
/tree 显示项目树
/pwd 查看当前路径
/open <file> 打开文件
/cat <file> 查看文件内容
/edit <file> 编辑文件
/diff 查看修改差异
/save 保存当前修改

3. Git 相关

指令 作用
/git status 查看 Git 状态
/git diff 查看 Git 修改
/git log 查看提交历史
/commit 自动生成 commit message
/push 推送代码
/pull 拉取代码
/branch 查看分支
/checkout <branch> 切换分支

4. AI 编程辅助

指令 作用
/explain 解释代码
/fix 修复错误
/refactor 重构代码
/optimize 优化代码
/review 代码审查
/test 生成测试
/doc 生成文档
/comment 添加注释
/convert 代码语言转换
/debug 调试问题

5. 项目分析

指令 作用
/analyze 分析项目结构
/architecture 分析架构
/dependencies 分析依赖
/security 安全检查
/performance 性能分析

6. 终端与运行

指令 作用
/run 运行项目
/build 构建项目
/shell 执行 shell 命令
/terminal 打开终端模式
/logs 查看日志

7. 上下文控制

指令 作用
/context 查看上下文
/memory 查看记忆
/forget 清除部分记忆
/summarize 总结当前会话

15. 一句话总结

Claude Code 最强的能力：

  • 理解整个项目
  • 重构复杂工程
  • 分析 AI 模型
  • 修复复杂 Bug
  • 自动生成工程代码

  • PyTorch
  • 时空预测
  • 深度学习
