先锋助理员管理系统 — 微信小程序用户手册

优化征求计划

  1. 文章下方有评论区，可通过评论区提供建议
  2. 直接将修改意见发送给总负责人（收取10%服务费）

一、如何打开小程序

  1. 打开微信，扫描小程序二维码
  2. 首次使用建议点击右上角 … → 添加到我的小程序

二、登录

2.1 首次登录（账号密码）

  1. 输入管理员提供的用户名和密码
  2. 点击 登录 按钮
  3. 登录成功后进入首页

2.2 绑定微信（推荐）

绑定后下次可直接微信一键登录，无需输入密码。

  1. 在登录页点击 微信一键登录 按钮
  2. 系统提示”该微信未绑定账号”→ 在弹出的窗口中输入用户名和密码
  3. 点击 绑定并登录
  4. 绑定成功后，以后点”微信一键登录”即可直接进入系统

2.3 修改密码

  1. 进入首页 → 点击 修改密码
  2. 输入旧密码、新密码、确认新密码
  3. 新密码至少 6 位
  4. 点击提交

三、首页

登录后进入首页，显示：

  • 数据概览：本月值班次数、待确认工资、已确认工资、本月工资
  • 功能菜单：我的值班表、我的工资条、物资统计、修改密码

四、我的值班表

查看未来两周的值班安排，支持签到和请假。

4.1 查看值班

  • 按日期时间线展示，标记”今天””明天””后天”
  • 每条值班显示时间段和地点
  • 今日值班会高亮显示

4.2 签到

签到分两种方式：

位置签到（当天值班）

  1. 点击值班卡片的 签到 按钮
  2. 选择 📍 位置签到
  3. 系统自动获取当前位置
  4. 在签到范围内 → 自动签到成功
  5. 超出范围 → 提示距离签到点 X 米，不自动提交
  6. 超出范围可改用”审核签到”

⚠️ 位置签到仅限值班当天使用。非当天的值班请使用审核签到。

审核签到

  1. 点击值班卡片的 签到 按钮
  2. 选择 👤 审核签到
  3. 提交后等待总负责人审核
  4. 审核通过后状态变为”已签到”

4.3 请假

  1. 点击值班卡片的 请假 按钮
  2. 填写请假原因（200 字以内）
  3. 提交后等待审核
  4. 状态说明：
状态 含义
请假待审 申请已提交，等待审批
已请假 请假已批准
请假驳回 请假被拒绝，可重新申请

五、我的工资条

查看和管理个人工资条。

5.1 查看工资条

  • 卡片列表展示，最新一条高亮（绿色边框）
  • 每条显示：年月、排班时长、实际时长、工资金额
  • 点击 详情 查看工时明细

5.2 确认工资条

工资条状态为”已发送”时：

  1. 点击 确认 按钮
  2. 确认无误后点击确定
  3. 状态变为”已确认”

5.3 提出疑问

对工资条有疑问时：

  1. 点击 提出疑问 按钮
  2. 填写具体疑问（200 字以内）
  3. 提交后负责人会收到反馈
  4. 状态变为”有疑问”

5.4 状态说明

状态 含义
草稿 负责人尚未发布
已发送 已发布，待你确认
已确认 你已确认无误
有疑问 你已提出疑问，等待处理

六、物资统计

  • 目前物资统计还在测试阶段，敬请期待。

查看物资统计信息（只读）。

  • 按分类展示物资项目
  • 查看各类物资的数量和状态

如需修改物资信息，请联系总负责人。

七、常见问题

Q：微信登录提示”该微信未绑定账号”？
A：按弹窗提示输入账号密码完成绑定，之后即可一键登录。

Q：位置签到提示”超出范围”？
A：说明你当前位置距离签到点较远。确认确实在值班地点后，改用”审核签到”。

Q：位置签到提示”仅限当天值班”？
A：位置签到只允许值班当天使用。如果是明天的值班，请用审核签到。

Q：签到/请假后状态没变化？
A：审核签到的状态更新需要负责人审批，刷新页面可查看最新状态。

Q：忘记密码怎么办？
A：请联系总负责人重置密码。

八、联系支持

如遇到系统问题或需要帮助，请联系总负责人。
